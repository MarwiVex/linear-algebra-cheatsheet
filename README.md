## English Overview
This repository contains a comprehensive cheatsheet for Linear Algebra, designed in LaTeX. It serves as a unified reference covering everything from fundamental vector spaces to advanced matrix factorizations and the fundamental theorem of linear algebra.

**Main Topics:**
* Vector Spaces and Views of Matrix Multiplication.
* The Four Fundamental Subspaces and Orthogonality.
* Systems of Equations, Elimination, and Determinants.
* Matrix Factorizations (LU, QR, Eigen, SVD).
* Eigenvalues, Projections, and Linear Transformations.

## Resumen en Español
Este repositorio contiene un formulario exhaustivo de Álgebra Lineal diseñado en LaTeX. Es una referencia unificada que abarca desde espacios vectoriales fundamentales hasta factorizaciones de matrices avanzadas y el teorema fundamental del álgebra lineal.

**Contenido Principal:**
* Espacios Vectoriales y Vistas de Multiplicación de Matrices.
* Los Cuatro Subespacios Fundamentales y Ortogonalidad.
* Sistemas de Ecuaciones, Eliminación y Determinantes.
* Factorizaciones de Matrices (LU, QR, Eigen, SVD).
* Valores Propios, Proyecciones y Transformaciones Lineales.
