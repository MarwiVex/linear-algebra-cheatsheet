# Linear Algebra Cheatsheet

This directory contains the English version of the cheatsheet, designed in LaTeX. It serves as an ideal reference material for students, engineers, and researchers in mathematics, physics, and computer science. The document concisely summarizes essential formulas, theorems, and computational pipelines necessary for linear algebra and matrix analysis.

---

## Core Content

* **Vector Spaces & Multiplication:** Axioms, geometric inequalities, and the five views of matrix multiplication.
* **Fundamental Subspaces:** Column space, nullspace, row space, left nullspace, and their orthogonal relationships.
* **Systems & Determinants:** Gaussian elimination, row echelon forms, matrix inverses, and properties of determinants.
* **Matrix Factorizations:** LU Decomposition, QR Factorization (Gram-Schmidt), Eigen-Decomposition, and Singular Value Decomposition (SVD).
* **Eigenvalues & Orthogonality:** Characteristic equations, Markov matrices, projection matrices, and least squares.
* **Linear Transformations & Special Matrices:** Change of basis, rotations, and properties of symmetric, positive definite, and orthogonal matrices.
* **Algorithms & Intuition:** Quick pipelines for common operations and conceptual guides to mathematical structures.

---

## Usage and Compilation

If you wish to build the PDF document (`Linear_Algebra_Cheat_Sheet.pdf`) directly from the source code (`linear_algebra_cheatsheet.tex`), you can use your preferred LaTeX environment or run the following command in your terminal using `pdflatex`:

```bash
pdflatex linear_algebra_cheatsheet.tex
```

*Note: Depending on your setup, you may need to compile the document twice to ensure all cross-references and formatting are generated correctly.*
